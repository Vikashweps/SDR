## Описание проекта

Проект по работе с Software Defined Radio (SDR) для приема и передачи сигналов.

## Этапы реализации

### 1. Инициализация и настройка SDR
- Инициализация SDR устройства
- Конфигурация параметров приемника/передатчика
- Настройка частоты, усиления и других параметров

### Работа с буфером
- Выделение и управление буферами для приема/передачи
- Обработка PCM данных
- Чтение и запись аудиофайлов

### 2. Передача аудиосигнала
- Формирование и передача аудиосигналов
- Работа с временными метками
- Обработка потоков данных в реальном времени

### 3.  Моделирование формирования и приема BPSK-сигналов. Дискретная свертка.
- Формирование модели BPSK на языке Python
- Реализации функции BPSK, апсемплинга, свертки на Си
- Подключение драйверов для SDR, работа с буферами

## Используемые технологии

- C/C++
- Python
