## Описание проекта

Проект по работе с Software Defined Radio (SDR) для приема и передачи сигналов. Реализует основные этапы обработки сигнала: от модуляции до демодуляции и визуализации.

## Этапы реализации
### 1. Модуляция и Кодирование
*   **BPSK / QPSK:** Классические схемы фазовой манипуляции с маппингом битов и апсемплингом.
*   **OFDM (Orthogonal Frequency-Division Multiplexing):** 
    *   Генерация OFDM-символов через IFFT.
    *   Добавление циклического префикса (Cyclic Prefix) для защиты от межсимвольной интерференции.
    *   Параллельная передача данных по множеству поднесущих.

### 2. Обработка сигнала (DSP)
*   **Согласованная фильтрация:** Применение КИХ-фильтров (FIR) для максимизации SNR на приеме.
*   **Синхронизация:** Алгоритмы символьной и частотной синхронизации для корректного декодирования потока.
*   **Дискретная свертка:** Оптимизированная реализация для моделирования канала связи.

### 3. Визуализация (GUI)
*   **Интеграция ImGui:** Легковесный графический интерфейс для отладки и мониторинга.
*   **Real-time графики:** 
    *   Спектральная диаграмма (Spectrum Analyzer).
    *   Созвездие сигналов (Constellation Diagram) для оценки качества модуляции.
    *   Временные диаграммы принятого сигнала.

### 4. Работа с оборудованием (SDR)
*   Интеграция с **SoapySDR** для управления устройствами (PlutoSDR, USRP).
*   Прямая работа с I/Q данными (In-phase/Quadrature).
*   Эффективное управление буферами приема/передачи для минимизации задержек.


## Используемые технологии
- C/C++
- Dear ImGui, ImPlot, Python - для отрисовки
- CMake, Make - сборка
- SoapySDR, FFTW 
- PCM audio format
